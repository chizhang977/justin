# 装饰模式

装饰模式和装饰器模式是同一个模式，只是中文叫法不同。

本站统一使用“装饰器模式”作为主文档名称，完整内容请阅读：

- [装饰器模式](/docs/method/dp/装饰器模式)

核心记忆：

```text
接口不变
外层包装
动态增强
可多层叠加
```

典型例子是 Java IO：

```java
InputStream inputStream = new BufferedInputStream(
        new FileInputStream("app.log")
);
```

`FileInputStream` 提供基础读取能力，`BufferedInputStream` 在外层增加缓冲能力。
